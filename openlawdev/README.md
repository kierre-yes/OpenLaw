# PH Law RAG Assistant

A simple legal research assistant for Philippine law.

This project uses Next.js, TypeScript, Supabase, PostgreSQL, and pgvector to support retrieval augmented generation for legal question answering.

## Goal

Help users ask questions about Philippine law and get grounded answers based on stored legal sources, chunks, and citations.

## Planned features

- Legal source storage
- Chunk based retrieval
- Vector search with pgvector
- Citation aware answers
- Chat history

## Stack

- Next.js
- TypeScript
- Supabase
- PostgreSQL
- pgvector
